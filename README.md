# Fashion-MNIST CNN Image Classifier

A custom Convolutional Neural Network (CNN) built with PyTorch to classify
clothing images from the Fashion-MNIST dataset.

## Overview
- Designed and evaluated a custom CNN architecture for 10-class image classification
- Achieved **93.5% test accuracy**
- Dataset: Fashion-MNIST (70,000 grayscale 28x28 images across 10 clothing categories)

## Tech Stack
- Python, PyTorch, NumPy, Matplotlib

## Results
- **Test Accuracy: 93.5%**
- Full evaluation details in [Report.pdf](Report.pdf)

## How to Run
1. Open `fashion-mnist-cnn.ipynb` in Google Colab or Jupyter
2. Run all cells
